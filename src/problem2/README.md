## Start project

### Requirements

- Node.js (18+)

### Install dependencies

```sh
npm i
```

### Start in development mode

```sh
npm run dev
```

### Start in production mode

Run the following command

```sh
npm run build
```

Then host the `dist` as a static website and always rewrites 404 request to index.html
